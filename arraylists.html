Array Lists

An ArrayList is a resizable array. When you declare an ArrayList, you don't have to specify the size of it. To use an ArrayList, you have to import java.util.ArrayList from the Java library. Here's an example of declaring an ArrayList of integers. ArrayLists can be useful when you do not know how many elements you are dealing with. 

import java.util.ArrayList; 
 
ArrayListnumbers = new ArrayList();
In the parenthesis, you can either leave it empty or put an integer. The integer represents the initial value of the ArrayList. Each time the ArrayList reaches its capacity, it automatically creates a new array double the capacity and copies all elements over, which can be take up a lot of memory.

To add elements to an ArrayList, we need to use the add() method. The add() method adds the element to the end of ArrayList. 

numbers.add(1); //adds integer 1 to end of arraylist
numbers.add(2); //adds integer 2 to end of arraylist
The ArrayList is now {1, 2}

To access elements of the ArrayList, you can use the get() method. 

int num1 = numbers.get(0); //gets integer 1
int num2 = numbers.get(1); //gets integer 2
Like an array, the indexes start at 0.

You can also change elements of an ArrayList using the set() method. 

numbers.set(0,3); //changes first number to 3
The ArrayList is now {3, 2}.

Like an array, you can use for loops and foreach loops to iterate through elements of an ArrayList. 

for (int i = 0; i < numbers.size(); i++) { 
    System.out.println(numbers.get(i)); 
}
The size() method gives the number of elements, so we use this to determine when to stop. We use the get() method and the index variable i to access each element and print it.

You can also use a foreach loop to directly access each element without using an index. 

for(int i : numbers) {
System.out.println(i); 
}
The variable i directly accesses each element of the ArrayList.

Example

An assignment where I used ArrayLists was when I was tasked to read multiple sentences from a file, shuffle them, and create 6 new random paragraphs. Since I didn't know how many sentences there were, I used an ArrayList. In addition, using an ArrayList was useful because it has a built in shuffle method called Collections.shuffle() so I was able to shuffle the ArrayList very easily. Below is a snippet of code when I stored the sentences from the file into the ArrayList and shuffled the ArrayList. 

ArrayList sentences = new ArrayList(); //creating the public ArrayList to store the sentences
   public void readFile() { //reading the sentences from the file into the ArrayList
      try {
         BufferedReader input = new BufferedReader(new FileReader("data.txt"));
         String nextWord = input.readLine();
         while((nextWord = input.readLine()) != null) {
            sentences.add(nextWord);
         }
         input.close();
      }
      catch(IOException e) {
      }
      Collections.shuffle(sentences); //shuffling the sentences
   }
