Recursion

Recursion is when a method calls itself to break complicated problems into simpler ones.

In recursion, there is always a base case that stops the recursion. The recursive method always works towards the base case. Recursion is useful for solving complex problems that are difficult to solve using iteration.

For example, if we want to find the sum of all positive integers up to a number, we can use recursion. 

public int sumR(int a) { //recursive method that finds the sum of all the positive integers up to the number 
    if(a == 0) {
        return 0;
    }
    return sumR(a-1) + a;
}
In the example above, the method keeps calling itself and adding the previous number until it reaches 0, which stops the recursion.

Example

I was tasked with an assignment that instructed me to remove all spaces in a string. This can be solved using iteration, but the code will be much shorter and simpler using recursion. My plan was to use the trim() method for each index of the string. 

public String compact(String word) { //recursive method that removes all spaces in a string
    if(word.length() == 1) {
        return word;
    }
    return word.substring(0,1).trim() + compact(word.substring(1));
}
If I were to use a loop for this problem, it would take many more lines of code.
