Arrays

An array stores a fixed number of elements of the same data type. Once you declare an array, the size cannot be changed. Here is an example of initializing an array of integers. 

int numbers[] = {1,2,3,4,5};
In this example, we initialize an integer array called numbers that contains the 5 integers, 1, 2, 3, 4, and 5. You can access each element of an array using its index, which starts at 0. For example, the first element, 1, is at index 0. The second element, 2, is at index 1. You can access the element by using numbers[index] 

int num = numbers[1]; //sets num to 2
You can also change an element of the array. 

numbers[0] = 2; //changing the first element to 2
You can also create an array of a size. 

int numbers[] = new numbers[5];
This array is initialized with default integer values (0).

To access each element of an array, we can use a for loop or a foreach loop.

For loop 
for (int i = 0; i < numbers.length; i++) { 
    System.out.println(numbers[i]); 
}
Foreach loop 
for(int i : numbers) {
  System.out.println(i);
}
The variable i is used to access each element of array. numbers.length gives us the size of the array, so we use this to determine when to stop. We stop before the size because the index starts at 0, so the last index would be 1 less than the length.

Arrays are useful because you can just create 1 array instead of creating many variables. It is also very easy to get elements from an array, using a for loop or using the index.

2D Arrays

A 2D array is an array of arrays. It stores data in the form of a grid. Below is an example of initializing a 2D array of integers. 

int numbers[][] = { {1, 2, 3}, {4, 5, 6} };
In this example, we initialize an integer 2D array called numbers that contains the 6 integers, 1, 2, 3, 4, 5, and 6. There are 2 rows and 3 columns. Each row contains 3 numbers. You can access each element of an array using its row and column, which both start at 0. For example, the first element, 1, is at [0][0]. The second element, 2, is at index [0][1]. You can access the element by using numbers[row][column] 

int num = numbers[1][1]; //sets num to 5
You can also change an element of a 2D array like in a regular array. 

numbers[1][1] = 0; //changes index 1,1 from 5 to 0
We can also create a 2D array of a size. 

int numbers[][] = new numbers[2][3];
The array is initialized with default integer values (0). The first perameter specifies the amount of rows, and the second specifies the amount of columns.

To iterate through the elements, we can use nested for loops. 

for (int i = 0; i < numbers.length; i++) { 
    for (int j = 0; j < numbers[0].length; j++) { 
        System.out.println(numbers[i][j]); 
    } 
}
We use the outer for loop to iterate over each row, and the inner for loop iterates over each column. We use the row variable i and the column variable j to access each element and print it.
